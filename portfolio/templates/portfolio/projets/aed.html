Interaco manual (teclado) com o utilizador - Standard Input
O programa a desenvolver dever correr de forma interactiva.
Ao arrancar o programa, o mesmo dever ficar escuta de input, que ser fornecido via
teclado / standard input (stdin). No Anexo II apresentado um exemplo de cdigo Java
para tratar da interaco com o utilizador.
O input ser uma String, podendo ocorrer uma das trs situaes seguintes:
O input a String QUIT;
O input representa uma query vlida (constituda pelo cdigo da pergunta e pelo
nmero correcto de variveis);
O input representa outra coisa - sendo, neste caso, invlido.
O que fazer em cada situao...
Se o input for a String QUIT, o programa dos alunos deve terminar imediatamente.
(Pode assumir que o QUIT ser sempre feito com todas as letras maisculas.)
Se o input for uma query vlida, o programa dos alunos deve:
1. calcular a resposta pergunta;
2. apresentar no ecr essa mesma resposta;
3. apresentar no ecr o tempo (em ms) que demorou a calcular a resposta;
4. ficar escuta da prxima query.
Se o input representa outra coisa qualquer, o programa deve:
1. apresentar no ecr a mensagem de erro Pergunta desconhecida. Tente novamente.
2. ficar escuta da prxima query.
Nota sobre a validade das queries:
Se uma query tiver como primeira componente um cdigo de uma pergunta padro conhecida,
ento garantido que o resto da query ser tambm vlido (formato, variveis, etc.).

Anexo II - Cdigo de suporte para Leitura de Dados a partir do Standard Input
A classe Scanner do Java, para alm de permitir ler ficheiros de texto, tambm permite fazer
leituras directas do teclado, usando o System.in (nome que representa o standard input - que
geralmente corresponde ao teclado).
As funes do Scanner, quando usadas para interaco com o System.in, so bloqueantes
- ou seja, que o programa vai estar parado enquanto o utilizador no introduzir o seu input.
De seguida apresenta-se uma possvel lgica para gesto da interactividade:
Scanner in = new Scanner(System.in);
String line = in.nextLine();
while (line != null && !line.equals("QUIT")) {
QueryResult resultado = perguntar(line);
System.out.println(resultado.valor);
System.out.println("(demorou " + resultado.tempo + " ms)");
line = in.nextLine();
}
